Mes Projets

Creation d'un site web de vente de Cosmétiques

Le projet consistait à développer un site web e-commerce spécialisé dans la vente de produits cosmétiques. Ce site permettant aux utilisateurs de parcourir, sélectionner et acheter une variété de produits de beauté tels que le maquillage, les soins de la peau, les parfums et les produits capillaires.

Creation d'une application de gestion d'Agence Bancaire

Ce projet visait à l'application pratique de nos compétences acquises en symfony. Cette applicaton gère une agence a travers un administrateur qui créait les clients ainsi que leurs comptes. le depot, le retrait, ainsi que le virement est possible.
